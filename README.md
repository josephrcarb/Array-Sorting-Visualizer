# Sorting Algorithm Visualizer

## Description
- Simple python program that utilizes pygame library to visually display an algorithms path to sorting an array.
  
## How to use.
```
  cd into ./Sorting
  
  python sorting.py
  or
  open sorting.exe
  
  click algorithm to display sorting 
  click home button to go back
```
