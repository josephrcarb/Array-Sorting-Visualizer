# Sorting-Visualizer

### Description

Simple python program that utilizes pygame library to visually display an algorithms path to sorting an array.

### How to use:

#### 1. Install Dependencies

```
pip install -r requirements.txt
```

#### 2. Run program

```
python sorting.py
```

or

```
python3 sorting.py
```

### Current Implemented Algorithms

- [x] Insertion Sort
- [x] Quick Sort
- [x] Merge Sort
- [x] Bubble Sort
- [ ] Heap Sort
- [ ] Shell Sort
- [ ] Comb Sort
